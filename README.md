## Getting started

1. clone this repository
```bash
git clone https://github.com/Ssttar/TechTrek.git
```
2. Install the dependencies 
 ```bash
 pnpm install
 ```
3. start the server 
  ```bash
 pnpm dev
  ```
4. open your website
  ```
  http://localhost:3000
  ```
5. To create a production version of your app:
```bash
pnpm build
```
